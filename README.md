# Link Simulator

Usage:

```bash
        25 +  cargo run -- \
        26 +    --space-binary /path/to/space_binary \
        27 +    --space-port 5001 \
        28 +    --ground-binary /path/to/ground_binary \
        29 +    --ground-port 5002 \
        30 +    --delay-ms 250 \
        31 +    --bit-error-rate 0.001 \
        32 +    --max-bandwidth 1000000 \
        33 +    --ip 127.0.0.1
        34 +  ```
